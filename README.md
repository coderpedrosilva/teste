# D'BUZIOS

Projeto institucional e catálogo digital da marca **D'BUZIOS**, com foco em moda praia, apresentação visual premium e navegação por categorias de produtos.

## Conceito visual

- estética minimalista  
- identidade moda praia premium  
- navegação leve com dropdown de categorias  
- experiência adaptada para desktop e mobile  

## Conceito

Definir a identidade da marca D'BUZIOS de forma elegante, simples e comercial
